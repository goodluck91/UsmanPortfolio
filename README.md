# Personal Portfolio - Usman Goodluck Anana

This is a simple **single-page personal portfolio website** created using HTML5 and CSS3. The website showcases my skills, interests, and contact information.

## Features
- Semantic HTML5 structure (header, nav, main, section, footer)
- Sections: About Me, Skills, Contact
- Responsive layout using CSS Flexbox
- Visual styling with 3 main colors
- Hover effects on navigation links and skill badges

## Technologies Used
- HTML5
- CSS3 (Flexbox)
- Google Fonts (Roboto)
